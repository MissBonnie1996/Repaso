REPASO GENERAL DE ELEMENTOS DE UNA PÁGINA WEB

Hay tres elementos importantes es una página web:

  • Archivo HTML para el cuerpo
  • Archivo CSS para el diseño
  • Archivo JavaScript para crear las funcionalidades de la página
